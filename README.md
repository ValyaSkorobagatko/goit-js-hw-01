# goit-js-hw-01
# Домашнє завдання №1 — JavaScript Basics

Репозиторій з розв'язанням задач першого модуля.

## Завдання

1.  **Задача 1. Замовлення дроїдів** (`task-1.js`) — розрахунок загальної вартості замовлення.
2.  **Задача 2. Доставка товару** (`task-2.js`) — перевірка можливості доставки в різні країни.
3.  **Задача 3. Ширина елемента** (`task-3.js`) — обчислення загальної ширини елемента з урахуванням `box-sizing: border-box`.

## Як запустити
Кожен файл можна перевірити, підключивши його в `index.html` або запустивши через консоль Node.js:
```bash
node task-1.js
node task-2.js
node task-3.js
